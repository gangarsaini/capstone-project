# 🎬 YouTube Clone (MERN Stack)

A full-stack YouTube Clone built using **React, Node.js, Express, and MongoDB** with features like authentication, channel creation, video upload, comments, likes/dislikes, and view tracking.

---

## 🚀 Features

### 🔐 Authentication
- User Register & Login
- JWT-based authentication
- Protected routes

### 📺 Video Features
- Upload YouTube videos
- Display videos on homepage
- View count tracking (unique per user/session)
- Like / Dislike system (toggle-based)

### 📢 Channel Features
- Create channel (one per user)
- Upload videos to your channel
- View your channel videos
- Delete videos (only by channel owner)

### 💬 Comments
- Add comment
- Delete comment
- (Optional) Edit comment

### 🔍 Search & Filter
- Search videos by title
- Filter by category

---

## 🛠️ Tech Stack

### Frontend
- React.js
- Tailwind CSS
- React Router
- Axios

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
- bcrypt (password hashing)
